# Sarala Dynamic Map Dashboard

Interactive PIN-level portfolio dashboard for Sarala loan data.

## What It Shows

- Map view by PIN code
- DPD trend
- Overdue amount trend
- Default OS trend
- VIIRS night-light index
- Flood vulnerability index
- Prior-year rainfall
- Filters for year, geography, purpose, subpurpose, and risk variables

## How To View

Open `index.html` in a browser for the main portfolio map.

Open `bureau_comparison.html` for the West Bengal 2025 Equifax bureau vs Sarala comparison dashboard.

Publish this folder with GitHub Pages to share both dashboards.

The dashboard is self-contained, but it loads Leaflet, Chart.js, and map tiles from public CDNs/OpenStreetMap, so internet access is required.
